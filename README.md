# City Casino

Top-down 2D city exploration game z slot machine + roulette mini-games.

**Stack:** Vite + React 18 + PixiJS v8 + GSAP + Zustand · Fastify + Drizzle + PostgreSQL · pnpm monorepo · Docker Compose.

## Quick start

```bash
pnpm install
docker compose up -d --build
open http://localhost:5173
```
